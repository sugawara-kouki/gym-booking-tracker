Technical Workflow Guide

1

Notification

施設からGmail宛に、抽選結果や仮予約完了の通知メールが送信されます。

2

Capture

Cloudflare Workersが定期実行によりGmail API経由で新着メールを検知・取得します。

3

Normalization

Hono上のロジックがメール本文を解析。予約情報を抽出し、D1 DBへ永続化します。

4

Presentation

保存されたデータはAPIを通じてダッシュボードに反映。ユーザーは一括管理が可能です。
